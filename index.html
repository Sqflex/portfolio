О Проекте

Проект был сделан с нуля за 4 месяца группой из четырех человек. Комплекс приложений - создание базы данных адресов предприятия, создание сайта для более простого заполнения адресов вручную и создание сайта для поиска ближайшего адреса по местоположению пользователя. Создание двух мобильных приложений (Android и IOS). Создание собственного API, выдающего json по определенным параметрам либо же со всеми адресами в правильном формате. Работа с API Яндекс карт и работа с приложениями Яндекс.Такси, GoogleMaps и AppleMaps. В мои обязанности входило проектирование, создание и оптимизация базы данных и создание структуры json-файла, а также создание сайта для заполнения и сайта для поиска своего участка по введенным данным (Область/район/город/улица/дом), которые по введенным данным также выбрасывали подсказки в формы по прохождению структурированного json-файла. По сути сайт для заполнения работал напрямую с базой данных, а сайт для поиска использовал зашифрованный json-файл для безопасности и отсутсвия SQL-инъекций.
